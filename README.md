### Install
> git clone git@github.com:hisea/dotvim.git .vim

> ln -s .vim/vimrc .vimrc

> ln -s .vim/gvimrc .gvimrc

> cd .vim

> git submodule init

> git submodule update

> mvim -u ~/.vim/configs/bundles.vim +PluginInstall +qall
